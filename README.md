# Practise git command

 update a changed file:
 ```
   git status 
   git add <changed file> 
   git status 
   git commit -m <description>
   git push origin master
```
delete a file local & remote
```
   git rm /file path/
   git status
   git commit -am <description>
   git status
   git push
```
